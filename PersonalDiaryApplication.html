Python project: Personal Diary Application

This Python application allows users to write and save diary entries using a GUI built with Tkinter.

Key Features:

  • Create Directory: Automatically creates a directory named with the current date.
  • Save Entry: Saves user input as a timestamped text file in the date-specific directory.
  • GUI Elements:
    • A Text box for writing entries.
    • Buttons to Save the entry or Quit the application.
  • Error Handling: Displays warnings or error messages using Tkinter's messagebox.

How It Works:

  1. Creates a directory named with the current date when the application starts.
  2. User writes an entry in the provided text box.
  3. Clicking Save creates a timestamped file in the directory and writes the entry to it.
  4. Handles empty entries or file system errors with user-friendly messages.

Usage:

Run the script in Python to open the GUI. Write your diary entry and click Save to store it.
